# 明镜 - 心理向自我成长小程序

<img src="./assets/明镜.png" alt="明镜图片" width="300">    

明镜是一款面向日常心理健康管理的AI小程序，通过创新性地结合语音情感计算、多模态情绪分析和个性化引导生成，解决了现有心理健康产品"重诊断轻陪伴"、"标准化回复缺乏个性"的长期痛点。

[![完成度](https://img.shields.io/badge/完成度-核心功能可用-green)](.) [![AI创新](https://img.shields.io/badge/AI创新-多模态情绪计算-blue)](.)

---

## 🎯 创新解决的核心问题

### 现有产品痛点分析
- **传统心理健康App**: 过度医疗化，普通用户使用门槛高
- **聊天机器人**: 标准化回复，无法提供个性化的情绪映照和行为洞察
- **情绪记录工具**: 单纯记录，缺乏智能分析和行动引导
- **心理咨询平台**: 成本高、等待时间长，无法满足即时情绪支持需求

### 突破性解决方案
明镜创新性地提出**"记录-映照-引导"三阶段AI陪伴模式**，将AI能力深度融入情绪健康管理的完整链路：

情绪表达 → AI情感计算 → 个性化映照 → 行动引导 → 模式洞察
↓           ↓           ↓           ↓           ↓
语音/文字   多模态分析   共情+客观   技能推荐   关系网络

## 🚀 AI创新能力亮点

### 1. 多模态情绪计算引擎
**创新点**: 首创语音情感特征 + 文本语义分析的融合计算
- **语音维度**: 集成百度ASR + 自研情感强度检测
- **文本维度**: 基于大语言模型的情绪细粒度识别（8维情绪 + 风险评分）
- **融合输出**: 生成个人情绪画像和风险预警

```javascript
// 核心AI能力示例
{
  "emotions": [
    {"name": "焦虑", "score": 0.73, "source": "voice+text"},
    {"name": "疲惫", "score": 0.61, "source": "voice"},
    {"name": "委屈", "score": 0.45, "source": "text"}
  ],
  "risk_score": 0.23,
  "voice_features": {"energy": 0.34, "pitch_variance": 0.12}
}
```

### 2. 个性化共情对话生成
**创新点**: 基于用户历史情绪模式的动态Prompt工程
- **角色适配**: AI根据用户表达方式自动切换"倾听者"、"分析师"、"教练"角色
- **情境感知**: 结合当前情绪状态和历史模式，生成个性化回应
- **功能卡触发**: 智能判断何时提供具体行动建议vs纯粹共情

### 3. 关系网络智能建模
**创新点**: 从情绪事件中自动抽取人物关系并可视化互动模式
- **实体识别**: 从用户叙述中提取人物、话题、情绪三元组
- **关系推理**: 基于共现频率和情绪关联度构建人际网络图
- **模式发现**: 识别关系中的情绪触发点和互动模式

### 4. 预测性干预系统
**创新点**: 基于情绪轨迹预测和主动干预触发
- **风险识别**: 多维度安全词检测 + 情绪轨迹异常检测
- **即时干预**: 自动触发呼吸练习、求助引导等救急措施
- **长期预防**: 通过行为模式分析提供个性化的情绪管理建议

## 📱 产品完成度展示

### ✅ 已完成核心功能

#### 阶段一：智能情绪记录
- **语音倾诉**: 实时录音 + ASR转写 + 情感特征提取
- **文字记录**: 多模板输入 + 实时情绪分析
- **技术实现**: 集成百度语音识别，支持暂停续录、实时转写

#### 阶段二：AI情绪映照  
- **多维分析**: 摘要、主题、人物、情绪8维评分、风险等级
- **个性化对话**: 基于LLM的共情回应 + 客观视角切换
- **安全守护**: 高危情绪检测 + 自动求助引导

#### 阶段三：行动引导闭环
- **智能推荐**: 根据情绪状态推荐个性化技能卡（呼吸法、沟通脚本等）
- **计时体验**: 可视化练习指导 + 完成度追踪
- **行动清单**: 与"今日"页联动，形成可执行的行动计划

#### 数据洞察可视化
- **情绪罗盘**: Canvas情绪云图 + 时间线归档
- **关系网络**: ECharts人物关系图 + 交互事件详情
- **行为模式**: 长期数据挖掘和趋势分析

- 抖音小程序开发者工具
